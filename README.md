# My website

Code for my website: <a href="l-omar-chehab.github.io" target="_blank">l-omar-chehab.github.io</a>.

### How to edit

Once this is downloaded into a local repository, run

```bash
# Run the website locally
bundle exec jekyll serve
```

and copy-paste the url that appears in the terminal into your internet browser. 

### Sources

This website is inspired by <a href="https://lena-voita.github.io" target="_blank">Lena Voita's website</a> and the <a href="https://themesinfo.com/penscratch-2-wordpress-theme-image-fu2w" target="_blank">PenScratch theme</a>.
